# Random-User-Directory-


A simple directory app using the [Random User API](https://randomuser.me/).

## Features
- Fetch random user profiles
- Display name, email, location, and photo
- Responsive grid layout

## Setup
1. Clone the repo.
2. Open `index.html` in your browser.
3. Or deploy to GitHub Pages.

## Deployment
- Push this repo to GitHub.
- Go to **Settings → Pages → Source → main branch → /root**.
- Your app will be live at `https://your-username.github.io/random-user-directory`.
